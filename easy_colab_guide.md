# Google Colab 파인튜닝 초간단 가이드

복잡한 설명은 빼고, **따라하기만 하면 되는** 단계별 가이드입니다.

## 1단계: 데이터 준비
먼저 내 컴퓨터에서 데이터셋을 만듭니다.
1. VS Code 터미널에서 다음 명령어를 실행하세요:
   ```bash
   python prepare_dataset.py
   ```
2. `training_data.jsonl` 파일이 생성되었는지 확인하세요.

## 2단계: Google Colab 열기
1. [Google Colab](https://colab.research.google.com/) 사이트에 접속합니다.
2. **새 노트**를 클릭합니다.
3. 상단 메뉴에서 **런타임 > 런타임 유형 변경**을 클릭합니다.
4. **하드웨어 가속기**를 **T4 GPU**로 선택하고 **저장**을 누릅니다. (필수!)

## 3단계: 파일 업로드
1. Colab 왼쪽 사이드바에서 **폴더 아이콘(파일)**을 클릭합니다.
2. 내 컴퓨터에 있는 `training_data.jsonl` 파일을 드래그해서 Colab 파일 목록에 넣습니다.
3. 업로드가 완료될 때까지 잠시 기다립니다. (파일이 보이면 성공)

## 4단계: 코드 실행
1. `colab_finetune_code.py` 파일의 내용을 **전체 복사**하세요.
2. Colab의 코드 입력창(셀)에 **붙여넣기** 하세요.
3. 왼쪽의 **재생 버튼(▶️)**을 누르세요.

## 5단계: 기다리기 & 다운로드
1. 코드가 실행되면서 라이브러리 설치 -> 모델 다운로드 -> 학습이 진행됩니다. (약 10~20분 소요)
2. 학습이 끝나면 `model_gguf` 폴더가 새로 생깁니다.
3. 그 안에 있는 `.gguf` 파일을 우클릭하고 **다운로드**하세요.

## 6단계: 사용하기
다운로드 받은 `.gguf` 파일을 Ollama에 넣어서 사용하면 됩니다!
```bash
ollama create my-law-model -f Modelfile
```
(Modelfile 내용은 이전 가이드 참고)
